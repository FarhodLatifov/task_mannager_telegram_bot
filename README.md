# Task Manager Telegram Bot

A powerful and user-friendly Telegram bot for managing your tasks directly from your chat interface.

## Features

- **Task Creation**: Add tasks with titles, categories, and priorities
- **Deadlines**: Set deadlines with an intuitive calendar interface
- **Attachments**: Attach photos or documents to your tasks
- **Progress Tracking**: Mark tasks as completed and track your progress
- **Search Functionality**: Quickly find tasks using search queries
- **Categorization**: Organize tasks into categories (Work, Personal, Shopping, etc.)
- **Multilingual Support**: Available in English and Russian

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your `BOT_TOKEN`
4. Run the bot: `python main.py`

## Usage

Start a conversation with your bot on Telegram and use the intuitive menu interface to manage your tasks efficiently.
